Personal Project · Feb 2026

Linux I²C Kernel Driver — SHT30 Sensor

Wrote a real, functional Linux kernel module in C from scratch targeting a Raspberry Pi on kernel 6.12.62, implementing a complete I²C character device driver for an Adafruit SHT30 temperature and humidity sensor. Combined two kernel subsystems in a single module: the I²C driver subsystem handles hardware matching via device tree and calls the driver's probe function to initialize the sensor, while the character device subsystem exposes a /dev/sht30 interface that userspace can read with standard Unix I/O.

Wrote a DTS device tree overlay compiled with dtc and loaded dynamically at runtime to describe the SHT30 at I²C address 0x44 on the Pi's I2C1 bus. Used of_match_table for device tree-based matching. The probe function validates sensor presence by querying its status register over raw I²C before registering the character device. Read calls trigger a measurement command via i2c_master_send, wait 15ms, then retrieve 6 bytes of raw data with i2c_master_recv. Raw 16-bit values are passed to userspace with copy_to_user, keeping data conversion in a userspace test application per the Linux "mechanism not policy" philosophy. Used devm_kzalloc for kernel memory allocation and file->private_data for per-open device state.

C Linux Kernel I²C Driver Character Device Device Tree Raspberry Pi copy_to_user

Formula FSAE · 2025–Present

FSAE Vehicle Wiring Harness

Designed the complete wiring architecture from scratch for LMU's Formula SAE car, integrating a Honda CBR600RR engine with a Haltech Elite 1500 ECU. Broke the system into three purpose-built sub-harnesses: a front harness serving cockpit sensors and dashboard, a rear harness driving cooling fans, fuel system, and pneumatic shifter, and an engine harness routing all powertrain sensors and actuators. Sensor suite includes coolant temp, MAP, throttle position, oil pressure, and neutral switch. Routed harnesses to avoid exhaust heat, and applied shielded cabling for the knock sensor to prevent EMI-induced signal noise. Designed FSAE-compliant shutdown circuit and a dual master-switch ignition system with cockpit override. Used RapidHarness to produce the full 60+ wire schematic. Design was awarded $1,700 in university funding.
