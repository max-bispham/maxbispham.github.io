32-bit ALU — Single Cycle CPU Design

Designed and implemented a 32-bit Arithmetic Logic Unit in SystemVerilog and Verilog as part of a single-cycle CPU design. Supported operations include addition, subtraction, logical shifts (left/right), and bitwise operations (AND, OR, XOR, NOT) with proper flag outputs (N, Z, C, V). Wrote a full Verilog testbench to verify correct functionality across all supported operations.

SystemVerilog Verilog Vivado ALU CPU Design Testbench
Analog IC Design · 2026

Discrete BJT Operational Amplifier

Designed and built a two-stage discrete op-amp using CA3046 NPN transistor array and MPSA-13 Darlington pair, meeting all specs (gain 400–500, Rid ≥ 20kΩ, Ro ≤ 200Ω). Applied Miller compensation for stability and designed an inverting amplifier configuration. Full flow: hand calculations → MultiSim simulation → hardware validation.

CA3046 Darlington MultiSim Miller Compensation Bode Plots
Engineering Tool · 2026

BJT Amplifier Design-Space Explorer

Built a custom Python tool to automate BJT amplifier design-space exploration for lab use. Given target specifications, the tool sweeps component values and calculates gain, input/output impedance, and bias point across hundreds of configurations, dramatically speeding up the design process. A practical example of using software to solve a real hardware engineering problem.

Python BJT Analog Design Automation Engineering Tool
